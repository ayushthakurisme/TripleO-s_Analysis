# TripleOS Sales Analysis

**Objective:** Analyze sales and customer data to identify the best-selling burger, returning customers, customer service feedback, and slowest day part.  

## Skills Used
- Excel (Pivot Tables, Charts, Dashboard)
- Data Analysis
- Data Visualization

## Dataset
- Simulated sales data for 10 days
- Columns: Date, Day Part, Burger Name, Quantity, Customer ID, Thumbs Up

## Key Insights
- Best-Selling Burger: Chicken strips
- Returning Customers: 40% of total customers
- Thumbs Up: 68% positive feedback
- Slowest Day Part: Dinner

## Dashboard
- Shows all key insights in charts and tables
- Includes Best-Selling Burger bar chart, Thumbs Up pie chart, Sales by Day Part column chart, and Returning Customers table

## How to Use
1. Open `TripleOS_Burger_Analysis_Dashboard.xlsx` in Excel
2. Explore the Pivot Tables and Charts
3. Review the dashboard for insights
